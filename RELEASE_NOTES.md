## New Features

### Settings Panel
A dedicated Settings modal replaces scattered toolbar controls. Access it via the gear icon in the toolbar.

Organized into four tabs:
- **Appearance** — Theme, language, expand tabs toggle
- **Editor** — Font family selection (presets + system fonts), line height, spellcheck
- **Code** — Code font family, word wrap
- **General** — Auto-save, version info, "What's New" link

### Font Selection with System Font Detection
Choose from curated font presets or pick any font installed on your system. The app detects available fonts across Windows, macOS, and Linux.

### What's New / Changelog
See the changelog for your current version inside the app — go to Settings > General > What's New. Update notifications also show a full rendered changelog.

### Expand Tabs Option
New setting in Appearance to let tab cards grow to fit the full folder + filename instead of truncating.
